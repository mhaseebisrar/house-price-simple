House Price Prediction 🏠

This is a simple Machine Learning project to predict house prices using the House Prices dataset.

Steps Performed:

- Loaded the dataset
- Cleaned the data (handled missing values)
- Selected numerical features
- Split data into training and testing sets
- Trained a Random Forest model
- Evaluated the model performance

Model Used:

- Random Forest Regressor

Result:

The model was trained successfully and produced a prediction score.

Tools & Libraries:

- Python
- Pandas
- Scikit-learn

How to Run:

1. Install required libraries:
   pip install pandas scikit-learn
2. Run the file:
   python main.py

Dataset:

House Prices Dataset (Kaggle)
